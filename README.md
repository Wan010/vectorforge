# VectorMagic - Advanced Image to Vector Converter

![VectorMagic Banner](assets/images/logo.svg)

A powerful, browser-based tool that converts raster images to scalable vector graphics (SVG) using advanced tracing algorithms. 100% client-side processing - no server required!

## 🚀 Features

### Core Conversion
- **Multiple Tracing Algorithms**: Auto-detection, Edge-based, Color-based, Posterize, and Sketch modes
- **Real-time Preview**: Live preview with split-view comparison
- **Advanced Controls**: Adjustable threshold, smoothness, noise reduction, and corner detection
- **Color Management**: Full color, grayscale, black & white, and limited color palettes

### Export Options
- **SVG Export**: Optimized vector output with minification options
- **PNG/JPG Export**: Convert vectors to raster formats at any resolution
- **Code Export**: Copy SVG code and generated CSS
- **Multiple Download Formats**: SVG, PNG, JPG with quality controls

### User Experience
- **Drag & Drop Interface**: Intuitive file upload with visual feedback
- **Preset Configurations**: Optimized settings for logos, sketches, photos, and artwork
- **Zoom & Pan**: Interactive preview with smooth navigation
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile

## 🛠️ Technology Stack

- **Frontend**: Pure JavaScript (ES6+), HTML5 Canvas, SVG
- **Processing**: Advanced image processing algorithms
- **Styling**: Modern CSS with Grid/Flexbox
- **No Dependencies**: 100% vanilla JavaScript
